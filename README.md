# My-MERN-STACK

## clone or Send Pull Request
```terminal
$ git clone https://github.com/amazingandyyy/mern-stack
```

# Usage(make it run on your machine)
### Client-side usage(PORT: 3000)
```terminal
$ sudo npm install webpack -s
$ nodemon
$ webpack -w
```
